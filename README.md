# cpp-modern-mastery

> **A hands-on C++ learning repository** — built from raw pointers all the way to C++20 concepts.  
> Every topic is implemented from scratch, documented with personal notes, and applied in real projects.  
> No copy-paste. No shortcuts. Just clean, modern C++.

---

## Why this repo exists

Most C++ tutorials either stop at the basics or jump straight into library usage without explaining what's happening underneath. This repo exists to bridge that gap — every concept is implemented manually, with comments explaining *why* the code is written the way it is, not just *what* it does.

---

## What's covered

| Phase | Topics | Status |
|-------|--------|--------|
| **Phase 1 — Foundations** | Pointers, references, manual memory, OOP, const correctness | ✅ Complete |
| **Phase 2 — Core Modern C++** | Smart pointers, move semantics, lambdas, Rule of Five, std::optional | 🔄 In progress |
| **Phase 3 — Advanced C++** | Templates, constexpr, C++17 features, C++20 concepts & ranges | 🔜 Upcoming |
| **Phase 4 — Expert Patterns** | RAII, TMP, concurrency, design patterns, coroutines | 🔜 Upcoming |
